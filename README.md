# SURE-Econometric-Analyzer
A desktop econometrics application built with Tkinter, Statsmodels, and Linearmodels to estimate and diagnose a two-equation Seemingly Unrelated Regression (SURE/SUR) system.
- **Joint Estimation**: Estimates joint SURE systems and provides separate OLS benchmarks.
- **Cross-Equation Diagnostics**: Computes residual covariance/correlation matrices.
- **Hypothesis Testing**: Performs Breusch-Pagan LM tests for residual independence and Jarque-Bera tests for normality.
- **Visualizations**: Generates residual correlation heatmaps and scatter plots.
- **Automated Reporting**: Exports a comprehensive multi-page econometric report to a single PDF document.
